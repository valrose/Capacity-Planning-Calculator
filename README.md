# Capacity Planning Calculator
Calculator designed to calcuate avaliable workforce capacity and compare against projected demand. 

## Overview

The Capacity Planning Calculator is a workforce planning tool designed to evaluate whether a team has sufficient available capacity to absorb new project demand.

The application calculates:

- Available team capacity
- Projected utilization
- Capacity gap
- Overall workforce health status

---

## Features

### Version 0.1

- Calculates available capacity based on employee allocations
- Calculates projected utilization after new demand is introduced
- Identifies capacity gaps
- Provides workforce health status:
  - Healthy
  - At Risk
  - Over Capacity

---

## Operational Context

Organizations frequently struggle to understand whether existing workforce capacity can absorb new demand.

This tool models workforce allocation, available capacity, and projected utilization to support staffing and planning decisions.
